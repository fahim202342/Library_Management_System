# Security Policy

## Supported Versions

This project follows a single-stream development model on the default branch.
Only the latest released or default-branch revision is supported.

---

## Reporting a Vulnerability

If you discover a security vulnerability, please **do not open a public issue**.

Instead, contact the project maintainer directly:

- **Email:** mf143416@gmail.com
- **GitHub:** [@fahim202342](https://github.com/fahim202342)

You will receive an acknowledgement within a reasonable time, and a fix or mitigation will be coordinated before any public disclosure.
